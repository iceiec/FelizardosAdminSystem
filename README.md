## 🎯 Project Overview

### Systems Included
1. **Pavilion Management** - Manage event dates, types, client information, bookings
2. **Pool Management** - Manage pool bookings, services, client information
3. **Court Management** - Manage basketball court schedules and bookings
4. **Maintenance Module** - Track maintenance requests across all three venues

### Tech Stack
- **Frontend**: React + Vite + React Router
- **Backend**: Express.js + Node.js
- **Database**: PostgreSQL
- **Authentication**: JWT + bcrypt
- **Other**: Axios, CORS, UUID
